・{% trans "Account related issues" %}
- {% trans "How can I bind my account ?" %}

1.當選取任一功能跳出以下畫面，請先登入line使Passcode獲得使用此帳號的權限

2.輸入帳號密碼做帳戶綁定

3.綁定後即會跳轉所選之功能頁面

- {% trans "I forgot my password, how can I reset it ?" %}

1.在帳號綁定頁面忘記密碼時，點擊忘記密碼按鈕

2.請先輸入帳號，之後點擊發送驗證碼

3.獲得驗證碼後輸入並送出，跳至輸入新密碼頁面，輸入兩次新密碼之後按下重設密碼送出

4.更新完成即會跳轉至帳戶綁定頁面

- {% trans "I don’t have a Passcode account, how do I register ?" %}

1.在帳號綁定頁面點擊註冊按鈕

2.填入欄位中資料按下下一步

3.點擊發送驗證碼，並在獲得驗證碼後填入欄位中，並按下確定送出

4.帳號註冊成功後即會跳轉至帳戶綁定頁面

・{% trans "Scan issues" %}
- {% trans "How to scan QR code ?" %}

1.在底下功能選單中選擇掃描

2.選擇掃描或是直接輸入使用者或地點名稱

備註：目前Line暫時關閉掃描QR Code功能，改以相機拍照方式上傳

3.掃描使用者之後會有3-1更新健康碼(掃描者需有更新健康碼權限)、3-2接觸登記、3-3訪客登記按鈕可以點選

3-1.更新健康碼可以選擇通過或未通過，代表將被掃描者的健康碼是否通過，送出結果後返回掃描頁面

3-2.接觸登記送出代表紀錄掃描者跟被掃描者有接觸過，送出結果後返回掃描頁面

3-3.訪客登記代表紀錄被掃描者訪問掃描者公司/學校，送出結果後返回掃描頁面

4.掃描地點之後會有4-1更新掃描人員狀態、4-2地點紀錄

4-1.更新掃描人員狀態代表將有更新健康碼權限之使用者開啟更新功能或關閉更新功能，送出結果後返回掃瞄頁面

4-2.地點紀錄代表紀錄掃描者到過此地點，送出結果後返回掃描頁面

- {% trans "How to show personal health code ?" %}

1.在底下功能選單中選擇顯示健康碼

2.健康碼狀態及個人QR Code會回應在對話欄中，健康碼狀態總共有"未填表、待測量、正常、高風險、危險"

未填表(黑色): 健康調查表未完成，須完成健康調查表

待測量(橘色): 待體溫測量

正常(綠色): 通過警衛/管理者更新健康碼

危險(紅色): 指健康調查表內容有體溫高於38.0以上或選項有勾選者，或是體溫過高，未通過警衛/管理者更新健康碼

・{% trans "Personal information settings" %}
- {% trans "Modify personal photo" %}

1.在底下功能選單其他中選擇個人資料設定

2.按下大頭貼右下方的相機

3.選擇拍照或是上傳照片修改大頭貼，選擇送出後即會更新大頭貼

- {% trans "Modify personal name company and remarks" %}

1.在底下功能選單其他中選擇個人資料設定

2.使用者可以更新使用者名稱、備註，輸入完畢後按下更新按鈕即會更新

- {% trans "Password reset" %}

1.在底下功能選單其他中選擇個人資料設定

2.使用者至此頁最下方，按下重設密碼按鈕進行重設密碼

3.點擊發送驗證碼，並在獲得驗證碼後填入欄位中，並按下確定送出

4.輸入兩次新密碼，送出後即修改密碼

・{% trans "Check in operation" %}
- {% trans "Location check in" %}

1.在底下功能選單中選擇位置打卡

2.如果手機有提示允許定位，請按允許同意passcode使用您目前的位置

3.待地圖定位完成，按下確定按鈕送出定位

3.打卡完成即會跳轉至歷史紀錄

- {% trans "Check-in activity annotation" %}

1.在底下功能選單中選擇位置打卡

2.如果手機有提示允許定位，請按允許同意passcode使用您目前的位置

3.待地圖定位完成，並在活動名稱、主辦單位、聯絡人聯絡電話中填入資訊，並按下確定按鈕送出定位

4.送出後歷史紀錄中會顯示打卡所填入的相關資訊

・{% trans "Company management" %}
- {% trans "How do I apply for a company" %}

1.在底下功能選單的其他中選擇公司管理

2.選擇公司申請

3.申請人為此line帳號綁定的人，填入公司/學校名稱及統一編號，按下確定送出申請

4.送出申請後會出現已經送出申請提示視窗

5.按下確定回到申請列表，申請會有"待審核、同意、取消"

待審核申請可由系統管理者同意或取消此申請，一般使用者可以主動取消待審核申請

6.使用者主動取消待審核申請會跳出提示視窗，按下確定即取消此申請

- {% trans "How do I enter the management background?" %}

1.在底下功能選單的其他中選擇公司管理

2.選擇後台連結

3.如有管理權限對話欄中會回應後台連結，並請使用電腦操作後台
